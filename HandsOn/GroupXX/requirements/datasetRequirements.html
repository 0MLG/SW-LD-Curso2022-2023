Dataset Requirements

Dataset Requirements: "calidad_aire_datos_mes.csv"

  1. R1. Data are in one of the selected smart city domains.

    Air quality.

  2. R2. Data are available as a CSV file

    calidad_aire_datos_mes.csv

  3. R3. Data have an open license so they can be published.

    Licencia Creative Commons 4.0 (CC BY 4.0). La licencia permite:

    • Compartir. Copiar y redistribuir el material en cualquier medio o formato.
    • Adaptar. Remezclar, transformar y crear a partir del material para cualquier finalidad, incluso comercial.

    Bajo la condición de Reconocimiento. Debe reconocer adecuadamente la autoría, proporcionar un enlace a la licencia e indicar si se han realizado cambios. Puede hacerlo de cualquier manera razonable, pero no de una manera que sugiera que tiene el apoyo del licenciador o lo recibe por el uso que hace.

  4. R4. Data can be easily linked with generic real-world entities.

    The are fiels that are real-world entities, like: provincia, municipio, estacion.

  5. R5. Documentation exists for the data (optional).

    Yes. There is pdf document (calidad_aire_datos_mes-doc) which describes the differents fields on the dataset.

  6. R6. Data can come from multiple data sources (optional).

    No. The data come from the AEMET (Agencia Estatal de Meteorología).
