R1. Data are in one of the selected smart city domains.

  • The application will offer relevant information about the air quality of the province of Madrid and its municipalities to the user.

R2. Data are available as a CSV file.

  • The dataset can be downloaded in CSV format here.

R3. Data have an open license, so they can be published.

  • The open license that allows us to reuse the data and documents related to the dataset can be verified via the "about open data" section of this website.

R4. Data can be easily linked with generic real-world entities.

  • The information inside the dataset could be linked to other entities such as municipalities inside Madrid and the province itself.

R5. Documentation exists for the data (optional).

  • The explicit documentation about the dataset is accessible here.

R6. Data can come from multiple data sources (optional).

  • Since there are multiple sensors located throughout Madrid, each one of these sensors could be counted as a different data source.
