Data source

Dataset filename: puntos-de-recarga.csv The data sets we are using is regarding all the charge station for electric vehicle in the city of Alcobendas. The available information is: - Owner, address, longitudes, latitudes, number of connectors, type of connectors, price, cables and adaptors, bookable, schedule, time limit
